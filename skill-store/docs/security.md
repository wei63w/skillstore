# 安全说明

当前阶段不接入真实支付、不保存真实银行卡、不提交真实密钥、不执行真实云端破坏性操作。
后续文件上传、支付模拟、权限配置和安全审核必须经过安全扫描与人工确认门禁。

交易闭环安全约束：

- 密码使用加盐哈希保存，接口响应不返回明文密码或密码哈希。
- 受保护接口使用 `X-Store-Token` 演示令牌识别用户，上传仅限创作者，审核仅限管理员，
  下单和支付仅限买家。
- Skill 上传限制 `.zip`、`.json`、`.txt`，大小不超过 1MB，空文件和不安全后缀会被拒绝。
- 上传文件只保存，不执行、不解压、不加载用户脚本。
- 模拟支付只接受成功或失败结果，不接入真实支付网关、不保存真实银行卡或支付密钥。
- 注册、上传、审核、下单和模拟支付写入审计日志，支撑后续追溯。
